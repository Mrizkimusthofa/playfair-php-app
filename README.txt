Playfair Cipher Demo (PHP + Bootstrap)
-------------------------------------

Extract this folder into your web server document root (e.g. XAMPP: htdocs/playfair-app).

Start Apache, then open:
http://localhost/playfair-app/public/

Usage:
- Enter passphrase (key) and message.
- Choose Enkripsi or Dekripsi then click Proses.
- Result and process log will appear on the same page.
- Click 'Play Steps' to animate the process log step-by-step (1 second per step).

Requirements:
- PHP 7.4+ with OpenSSL extension (no DB needed).
